Webプログラミングの演習課題

5420003 中村啓太郎

head要素

  1. 説明
    2. headの役割は主に検索エンジンにサイトの情報を提供したりすることである。またCSSやJsなどの外部ファイルを読み込んだりもする。
  2. 例
    3. <head>
    meta charset="utf-8"
    <title> 演習2 </title>
    <link href="styles/style.css" rel="stylesheet">
    </head>
  3. その結果
    4. タブの部分に「演習2」と表示される。またstyle.cssを読み込むことができる。

title要素

  1. 説明
    2. titleとはサイトにタイトルをつけるタグである(このサイトでいう演習2)
  2. 例
    3. <title> 演習2 </title>
  3. その結果
    4. タブの欄に「演習2」と表示され、検索結果の欄のURLの上にタイトルとして表示される。

html要素

  1. 説明
    2. <html>はこの文書がHTML文書であることを示すタグである。<head>や<body>などをhtml、/htmlでくくる。
  2. 例
    3. <html>
    <head>
    <title> 演習2 </title>
    <head>
    <body>
    本文
    </body>
    </html>
  3. その結果
    4. タブやタイトルに演習2と記され、Webページ上には本文と記される。
